## Installation

This is a Progressive Web App (PWA). You can:

1. **Use it in your browser** - Just visit the URL
2. **Install on your phone** - Tap "Add to Home Screen" for a native app experience
3. **Use offline** - Works even without internet after first visit

## Tech Stack

- Vanilla JavaScript (no frameworks!)
- HTML5 & CSS3
- Service Workers for offline support
- LocalStorage for data persistence
- Chart.js for analytics

## Development

To run locally:

1. Clone this repository
2. Open `index.html` in your browser
3. Or use a local server:
   ```bash
   npx serve
   ```
